# f-29: Implement locations command

- **Milestone:** M2
- **Complexity**: Low

## Description
Add a `kley locations` command that shows where the kley store and registry are located on the filesystem. Useful for debugging and understanding the tool's setup.

## Acceptance Criteria
1. `kley locations` prints the kley home directory path (`~/.kley`).
2. `kley locations` prints the registry file path (`~/.kley/registry.json`).
3. `kley locations` prints the packages store path (`~/.kley/packages`).
4. Output is clear and human-readable.
