# f-30: Add progress indicators

- **Milestone:** M2
- **Complexity**: Low

## Description
Add visual progress indicators for long-running operations like `publish --push` (which may update multiple projects) and `install` (which may run a package manager install).

## Acceptance Criteria
1. `publish --push` shows which project is being updated (e.g., "Updating project-a... ✔").
2. `install` shows a spinner or progress message while the PM install is running.
3. Progress output does not interfere with normal output or error messages.
4. Works correctly in terminals that do not support ANSI codes.
